# 基于 Tone.js 与 React 的 Demo

[Tone.js](https://yarnpkg.com/en/package/tone) 是一个制作合成音乐的 web 语音框架

[React.js](https://reactjs.org/) web 框架

项目基于 React 官方提供 `create-react-app` 脚手架搭建

## run code

1. yarn
2. yarn start
